# Education Blog posts

Add new education articles as Markdown or HTML files in this folder. Each post should include front matter like:

```yaml
---
title: "Post title"
date: 2026-07-01
author: "Author name"
summary: "One-sentence summary."
topics:
  - Topic one
  - Topic two
math: true
reading_time: "10 min read"
---
```

The `/education-blog/` page automatically lists posts from this collection by date.
